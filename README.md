# img2bag

## 1.img转bag，单topic
### 指定image路径, 生成的bag名,时间戳路径, 然后运行.
```
python img2bag.py /your_directory/images/ output.bag /your_directory/times.txt
```

## 2.img转bag，俩topic
### 指定image路径, 生成的bag名,时间戳路径, 然后运行.
```
python img2bag_Stereo.py /your_directory/images/ output.bag /your_directory/times.txt
```
